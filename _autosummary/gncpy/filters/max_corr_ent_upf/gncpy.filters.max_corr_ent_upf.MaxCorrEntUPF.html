property meas_likelihood_fnc

A function that returns the likelihood of the measurement.

This has the signature f(y, y_hat, *args) where y is the measurement as an Nm x 1 numpy array, and y_hat is the estimated measurement.

Notes

This represents \(p(y_t \vert x_t)\) in the importance weight

\[w_t \propto \frac{p(y_t \vert x_t) p(x_t \vert x_{t-1})}{q(x_t \vert x_{t-1}, y_t)}\]

and has the assumed form \(\mathcal{N}(y_t, R)\) for measurement noise covariance \(R\).

Returns:

function to return the measurement likelihood.

Return type:

callable

property proposal_fnc

A function that returns the probability for the proposal distribution.

This has the signature f(x_hat, x, y, *args) where x_hat is a gncpy.distributions.Particle of the estimated state, x is the particle it is conditioned on, and y is the measurement.

Notes

This represents \(q(x_t \vert x_{t-1}, y_t)\) in the importance weight

\[w_t \propto \frac{p(y_t \vert x_t) p(x_t \vert x_{t-1})}{q(x_t \vert x_{t-1}, y_t)}\]

and has the assumed form \(\mathcal{N}(\bar{x}_{t}, \hat{P}_t)\)

Returns:

function to return the proposal probability.

Return type:

callable

property transition_prob_fnc

A function that returns the transition probability for the state.

This has the signature f(x_hat, x, cov) where x_hat is an N x 1 numpy array representing the propagated state, and x is the state it is conditioned on.

Notes

This represents \(p(x_t \vert x_{t-1})\) in the importance weight

\[w_t \propto \frac{p(y_t \vert x_t) p(x_t \vert x_{t-1})}{q(x_t \vert x_{t-1}, y_t)}\]

and has the assumed form \(\mathcal{N}(f(x_{t-1}), P_t)\) for the covariance \(P_t\).

Returns:

function to return the transition probability.
