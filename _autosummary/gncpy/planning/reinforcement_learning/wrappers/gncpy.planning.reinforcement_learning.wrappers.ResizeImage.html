ResizeImage

class ResizeImage(env=None, n_rows=84, n_cols=84, key='img')[source]

Bases: ObservationWrapper

Wrapper for resizing an image.

This can also work with resizing an image in a dict space. Assumes the image is H x C x 3.

__init__(env=None, n_rows=84, n_cols=84, key='img')[source]

Initialize an object.

Parameters:

  • env (gym environment, optional) – Environment to wrap. The default is None.

  • n_rows (int, optional) – final number of rows. The default is 84.

  • n_cols (int, optional) – final number of columns. The default is 84.

  • key (string, optional) – key in the dictionary space corresponding to the image. The default is ‘img’.
