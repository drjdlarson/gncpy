CurvilinearMotion

class CurvilinearMotion(**kwargs)[source]

Bases: NonlinearDynamicsBase

Implements general curvilinear motion model in 2d.

This is a slight variation from normal NonlinearDynamicsBase classes because it does not use a list of continuous functions but instead has the state and input matrices coded directly. As a result, it also does not use the control model attribute because it is hardcoded in the get_input_mat() function. Also, the angle state should be kept between 0-360 degrees.

Notes

This implements the following system of ODEs.

\[\begin{split}\begin{align} \dot{x} &= v cos(\psi) \\ \dot{y} &= v sin(\psi) \\ \dot{v} &= u_0 \\ \dot{\psi} &= u_1 \end{align}\end{split}\]

See [14] for details.
